# 武器一覧

## 概要
3属性 × 2種 = 6武器
各武器は固有のコンボノード群とプレイスタイルを持つ

---

## 斬属性

### 大剣

| 項目 | 内容 |
|------|------|
| 属性 | 斬 |
| リーチ | 長い |
| 攻撃速度 | 遅い |
| ダメージ | 高い |
| 最大コンボ数 | 3 |

#### 特徴
- 大振りなモーション。コンボ1ノードあたりの時間が長い
- 硬直が大きく隙だらけになるが、当たった時のダメージが大きい
- 広範囲の攻撃判定を持つノードが多い

#### コンボ傾向
- ノード数は少なめだが1ノードの威力が高い
- 位置遷移が大きい（上→下、左→右など対角的な動き）
- コンボ失敗時のリスクが高い（長い硬直）

---

### 短剣

| 項目 | 内容 |
|------|------|
| 属性 | 斬 |
| リーチ | 短い |
| 攻撃速度 | 速い |
| ダメージ | 低い |
| 最大コンボ数 | 5 |

#### 特徴
- 小回りの効く素早いアクション
- リーチが短い分、敵に密着して戦う必要がある
- 背面から攻撃するとダメージ増加（固有補正）

#### コンボ傾向
- ノード数が多く、回転率の高いコンボが可能
- モーション時間が短いため入力受付窓も短い（操作精度が求められる）
- 位置遷移が小さい（隣接位置への移動が多い）

---

## 壊属性

### 槌

| 項目 | 内容 |
|------|------|
| 属性 | 壊 |
| リーチ | 中程度 |
| 攻撃速度 | 遅い |
| ダメージ | 高い |
| 最大コンボ数 | 3 |

#### 特徴
- 振り回しによるコンボは自身の軸も不安定になり、武器の勢いに持っていかれる挙動
- 叩きつけノードで衝撃波を発生させ、範囲攻撃が可能
- 一部ギミック（シーソー等の重量系）に使用できる可能性あり

#### コンボ傾向
- 振り回しによる慣性を感じるモーション
- 衝撃波ノードはコンボの締めとして強力だが、後隙が非常に大きい
- 大剣と同様にハイリスク・ハイリターン

---

### 棍

| 項目 | 内容 |
|------|------|
| 属性 | 壊 |
| リーチ | やや長い |
| 攻撃速度 | 速い |
| ダメージ | 低〜中 |
| 最大コンボ数 | 5 |

#### 特徴
- 短剣同様に小回りが効く
- リーチは短剣より少し長い
- バランス型の壊属性武器

#### コンボ傾向
- 回転や連打系のノードが多い
- 位置遷移がスムーズで繋ぎやすい
- 安定した立ち回りが可能

---

## 突属性

### 槍

| 項目 | 内容 |
|------|------|
| 属性 | 突 |
| リーチ | 非常に長い |
| 攻撃速度 | 中程度 |
| ダメージ | 中程度 |
| 最大コンボ数 | 4 |

#### 特徴
- リーチが非常に長く、安全な距離から攻撃可能
- 直線的な攻撃のみで、横方向や後方への攻撃判定がない
- 側面や背後からの敵の接近に弱い

#### コンボ傾向
- 前方向の位置遷移が中心（前→前の連突き等）
- 横方向の位置を経由するノードが少ない
- 距離管理がプレイヤースキルとして重要

---

### 銃

| 項目 | 内容 |
|------|------|
| 属性 | 突 |
| リーチ | 超長距離 |
| 攻撃速度 | 単発（連射不可） |
| ダメージ | 低い |
| 最大コンボ数 | 1（コンボなし） |

#### 特徴
- 圧倒的な射程と弾速
- 反動により連射不可
- ダメージは低い

#### コンボ
- コンボなし（単発攻撃のみ）
- 攻撃後は長めの硬直（反動回復）

#### 要検討事項
- コンボが存在しないことでゲーム性としてどう位置づけるか
  - 案1：コンボの起点・繋ぎとして他武器との切り替えコンボに組み込む
  - 案2：射撃後に別武器へ即座に切り替え可能にし、コンボの「始動」として使う
  - 案3：特殊弾（属性弾、状態異常弾等）で差別化し、コンボ不要な価値を持たせる
  - 案4：そもそも銃を武器枠から外し、サブウェポン/道具枠にする

- スタン入れるかなって思ってます

---

## 武器間バランス方針

- 高ダメージ × 低速度（大剣、槌）：ハイリスク・ハイリターン
- 低ダメージ × 高速度（短剣、棍）：ローリスク・ローリターン + 手数で稼ぐ（接近するリスクはあり）
- 中ダメージ × 中速度 + 射程（槍）：距離管理型
- 低ダメージ + 超射程（銃）：位置づけ要検討
