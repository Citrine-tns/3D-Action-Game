# 武器スイングシステム

## 概要
武器の振りモーションをコンボノードの始動位置・終端位置に基づいて生成する
アニメーションファイルは使わず、コードで回転を制御する

---

## オブジェクト階層

```
Player
└── WeaponPivot（空オブジェクト。右手の位置に配置）
    └── [武器名]（WeaponHolder が自動生成）← WeaponSwing が回転を制御する対象
        ├── Model（仮モデル Cube。見た目のみ）
        └── Hitbox（BoxCollider。ダメージ判定）
```

- WeaponPivot が回転の原点（手元）
- [武器名] を回転させると、Model と Hitbox が一緒に振られる

---

## 武器の軸

仮モデルは modelOffset = (0, Y, 0) で上方向に伸びる
つまり武器の「刃先方向」= ローカル Y+ 軸

```
      ↑ Y+（刃先方向）
      |
      |  [武器モデル]
      |
  Pivot（手元）──→ X+
```

---

## NodePosition と回転角度の対応

Pivot を原点として、武器の刃先がどの方向を向くかで決まる

| NodePosition | 刃先の方向 | 回転 (Euler) | 説明 |
|---|---|---|---|
| Upper | 上 | (0, 0, 0) | デフォルト。刃先が真上 |
| Lower | 下 | (180, 0, 0) | X軸180°。前方経由で下へ |
| Left | 左 | (90, 90, 0) | 水平面で左を向く |
| Right | 右 | (90, -90, 0) | 水平面で右を向く |
| Front | 前方 | (90, 0, 0) | 刃先が正面 |

### 振りの軌道

Upper → Lower（振り下ろし）は前方経由で下へ:
```
    ↑ 上（構え）
    |
    Pivot ----→ 前方（通過）
    |
    ↓ 下（振り終わり）
```

Left → Right（横薙ぎ）は水平面で前方を横切る:
```
   左 ← Pivot → 右
        ↑前方
```

## 状態ごとの挙動

```
[Idle]
  武器は待機角度（例: 少し斜めに構える）で静止
  Hitbox: OFF

  ↓ 攻撃入力

[Active]
  1. 武器を瞬時にノードの始動位置へ移動
  2. motionDuration をかけて終端位置へ回転
  Hitbox: ON（この間に敵と接触すればダメージ）

  ↓ motionDuration 経過

[Recovery]
  武器は終端位置で静止
  Hitbox: OFF

  ↓ 入力あり → 次ノードの Active へ
  ↓ 入力なし or 最終ノード → Idle へ

[Idle に戻る]
  武器が終端位置から待機角度へ滑らかに戻る
```

---

## Hitbox（当たり判定）

- Hitbox は武器オブジェクトの子なので、武器と一緒に回転する
- reach に応じて Hitbox のサイズを設定（武器のローカル Y 方向に伸ばす）
- Active 中のみ有効。振りの軌道上で敵に接触したらダメージ

```
  reach = 2          reach = 4
  ┌──┐               ┌──┐
  │  │               │  │
  │  │               │  │
  │  │               │  │
  └──┘               │  │
  Pivot              │  │
                     │  │
                     └──┘
                     Pivot
```
